# ML Projects 🤖

Machine Learning projects built from scratch 
using Python and scikit-learn.

## Projects

### 1. Salary Prediction — Linear Regression
- Predicts salary based on years of experience
- Algorithm: Linear Regression
- Tools: Python, NumPy, scikit-learn

### 2. Pass/Fail Prediction — Logistic Regression
- Predicts whether a student passes or fails
- Based on hours of study
- Algorithm: Logistic Regression
- Tools: Python, NumPy, scikit-learn

### 3. Loan Approval Prediction — Decision Tree
- Predicts whether a loan gets approved or rejected
- Based on income and credit score
- Algorithm: Decision Tree Classifier
- Tools: Python, NumPy, scikit-learn
## Skills Used
- Python
- NumPy
- scikit-learn
